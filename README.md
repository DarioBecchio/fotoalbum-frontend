# 📷 Foto Album – Frontend (Vue.js + Vite)

Frontend di un'applicazione per la gestione di un album fotografico, sviluppato con Vue.js e Vite. Fa parte del progetto conclusivo del corso Full-Stack Web Development di Boolean Academy, in coppia con [fotoalbum-backend](https://github.com/DarioBecchio/fotoalbum-backend).

## Funzionalità

- Visualizzazione dell'elenco delle foto recuperate dal backend
- Dettaglio della singola foto
- Comunicazione con le API REST esposte dal backend Laravel

## Tecnologie utilizzate

- Vue.js 3
- Vite
- Chiamate AJAX/Fetch verso API REST

## Cosa ho imparato

- Costruire un'interfaccia che consuma un'API REST costruita da me stesso (backend Laravel)
- Gestire routing e componenti in un'applicazione Vue strutturata
- Separare in due repository distinti frontend e backend, come in un progetto reale

## Come avviarlo in locale

```bash
git clone https://github.com/DarioBecchio/fotoalbum-frontend.git
cd fotoalbum-frontend
npm install
npm run dev
```

> Richiede il backend attivo: vedi [fotoalbum-backend](https://github.com/DarioBecchio/fotoalbum-backend) per le istruzioni.
